# CalmCompanion

A personal therapy companion built with React, Vite, TailwindCSS, and Framer Motion.  

## Features
- Chat with supportive assistant
- Mood tracking
- Journal & insights
- Data export/import
- Light/dark theme toggle

## Local Setup
```bash
npm install
npm run dev
```
